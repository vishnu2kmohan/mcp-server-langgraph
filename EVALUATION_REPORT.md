
**Workaround**: Used `uv pip` instead of `uv sync`
**Long-term Fix**: Either:
- Pin to infisical-python==2.3.5 (has compatible wheels)
- Make Infisical optional dependency
- Use conditional dependencies for ARM64 vs x86_64
